Um Protocolo é um conjunto de regras que dita o formato e ordem das mensagens trocadas entre uma ou mais entidades na [[Internet]], assim como as ações que acontecem nessa troca. 



Na prática, é uma sequência de passo requeridos para conectar dispositivos. Por exemplo, no protocolo HTTP, a forma como você requisita uma página HTML é mais ou menos assim:

- Computador pessoal solicita conexão (requisição/request 1)
- Host da página responde com uma permissão de conexão (resposta/response 1)
- Computador pessoa requisita rota API que retorna a página (requisição/request 2)
- Host da página devolve o arquivo HTML renderizado no navegador (resposta/response 2)

Existem muitos protocolos. Geralmente, cada tipo de comunicação/conexão possui suas regras específicas (protocolos)
